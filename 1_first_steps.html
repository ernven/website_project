First Steps: Resources which were useful for me

Official website for the World Wide Web Consortium

From here you can find information on the standards that govern design for the Web, like the ones we are learning like HTML and CSS (and soon JavaScript) but also multimedia and security features, for example. Here you can read up on the basis for the language. I have used this resource to learn about specifying the language of the document, when we were learning to use the HTML validator.

W3C HTML Validator

This is the validator itself, essential for checking we are writing proper HTML code.

W3C CSS Validator

This is the validator for CSS files/code.

W3Schools HTML tutorial

Very useful resource for learning about HTML elements and tags. I have learn here how to use many tags, like <div> and how you use them (which attributes do they take and the proper syntaxis).

Mozilla's CSS reference

Good source for CSS. You can browse all the elements, but also see basic rules and concepts (so not just reference).

StackOverflow

This one is technically not from the OSE Course page, but it's an amazing resource which I use for any kind of programming doubts and troubleshooting, and from where I've learnt a lot! Most often it includes great explanations.
